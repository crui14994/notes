# BaseFn

## 空间两点距离计算函数 getSpaceDistance

- @param

  positions(type:Array) : 两点的坐标数组

- @returns

  distance ：两点距离。如：

  ```js
  let distance = CM.BaseFn.getSpaceDistance(positions);
  ```

## 计算多边形面积 getArea

- @param

  points(type:Array) : 多边形点位的坐标数组

- @returns

  area ：面积 。如：

  ```js
  let area = CM.BaseFn.getArea(points);
  ```
