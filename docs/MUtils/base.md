#### FullScreen 全屏

- @param

| 参数名  | type | 描述                 | 默认值 |
| ------- | ---- | -------------------- | ------ |
| element | Dom  | 需要全屏的 html 元素 | -      |

```js
let dom = document.getElementById("id");
mUtils.mUtils(dom);
```

#### fullExit 退出全屏

- @param

| 参数名  | type | 描述                 | 默认值 |
| ------- | ---- | -------------------- | ------ |
| element | Dom  | 需要退出全屏的html元素 | -      |

```js
let dom = document.getElementById("id");
mUtils.fullExit(dom);
```
